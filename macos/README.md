# Harbr on macOS

See [docs/building-on-macos.md](../docs/building-on-macos.md) for the full setup guide.

## Quick Start

From the `macos/` folder inside the cloned repo:

```bash
chmod +x Step1-Prerequisites.sh Step2-Build.sh

# Install prerequisites (safe to re-run)
./Step1-Prerequisites.sh

# Open a new terminal, then build
./Step2-Build.sh
```
